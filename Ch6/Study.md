# 6장. 반복문

## 6.1 반복문의 개념

**반복문**이란?
특정 작업을 여러 번 반복적으로 실행할 때 사용하는 명령문이다.

**사용 목적:**

* 데이터 목록을 반복 처리
* 같은 작업을 정해진 횟수만큼 반복
* 조건을 충족할 때까지 반복 수행

**C언어에서 제공하는 3가지 반복문:**

* `while문`
* `do-while문`
* `for문`

**확인문제**
C언어에서 제공하는 반복문은 **(while문)**, **(do-while문)**, **(for문)** 이다.

---

## 6.2 while문

### 개념

`while문`은 주어진 **조건이 참인 동안** 특정 코드 블록을 반복적으로 실행한다.

```c
int a = 0;
while(a < 10) {
    printf("%d", a);
    a++;
}
```

### 동작 과정

```c
int count = 0;
while(count < 5) {
    printf("count: %d\n", count);
    count++;
}
```

* `count = 0` 초기화
* 조건식 `count < 5` 평가
* 조건이 참이면 내부 코드 블록 실행
* 실행 후 증감하고 다시 조건식 평가 반복
* 조건식이 거짓이 되면 반복문 종료

**주의:** 무한 루프 방지를 위해 적절한 종료 조건 설정 필요
**참고:** 윈도우 환경에서 입력 종료는 `Ctrl + Z`

**확인문제**

while문의 동작 순서:

1. **d**: 조건식 평가
2. **c**: 조건이 참이면 코드 블록 실행
3. **b**: 코드 실행 후 조건식 재평가
4. **a**: 조건이 거짓이면 반복문 종료

## 6.4 for문

### 개념

조건이 참인 동안 특정 코드 블록을 반복적으로 실행하는 반복 제어문

```c
for(초기화; 조건식; 증감식) {
    // 실행 코드
}
```

### 구성요소

* **초기화**: 반복 제어 변수 선언 및 초기값 설정
* **조건식**: 반복 수행 여부 판단 (참일 시 실행)
* **증감식**: 반복 변수 값 변화

### 예시

```c
int i;
for(i = 0; i < 10; i++) {
    printf("%d\n", i);
}
```

**실행 순서:**

1. `i = 0`으로 초기화
2. `i < 10` 조건 확인
3. 조건이 참이면 코드 블록 실행
4. 증감식 `i++` 실행
5. 다시 조건 확인 → 반복

---

### 다중 for문

`for문` 안에 또 다른 `for문`이 있는 구조

```c
int i, j;
for(i = 2; i < 10; i++) {
    for(j = 1; j < 10; j++) {
        printf("%d * %d = %d\n", i, j, i * j);
    }
    printf("\n");
}
```

**구구단 예제**

* 바깥 반복문 `i`: 2단 \~ 9단
* 내부 반복문 `j`: 곱해지는 수 1 \~ 9

각 단마다 곱셈 결과를 출력하며, 각 단이 끝날 때 줄바꿈을 해줌.

**별 출력 예제**
```#include <stdio.h>

int main()
{   
    int i, j;
    
    for(i = 0; i <= 5; i++){
        for(j = 0; j < i ; j++){
            printf("*");        
        }
        printf("\n");
    }
    return 0;
}
코드 설명: i가 출력되는 층의 수고 5단까지 결과를 출력할것임. j는 출력될 별의 수고, 줄 번호당 출력되는 별의 수가 줄번호보다 1개 적게 출력되어야 하므로
j < i를 조건문으로 넣었고, j를 후위연산하여 다음 출력층에서 나올 *의 수를 한개 늘려준다.
```

***별 출력 예제(우측 정렬)**
```
#include <stdio.h>

int main()
{   
    int i, j;
    
    for(i = 0; i <= 4; i++){
        for(j = 0; j < 4-i ; j++){
            printf(" ");        
        }
        for(j=0;j<=i;j++){
            printf("*");
        }
        printf("\n");
    }
    return 0;
}
코드 설명: i는 0부터 4층까지 총 5단을 출력하는 형태이며, 4-i개의 공백을 좌측부터 i=0층부터 출력한다, 그리고 이걸 출력하는 조건하에, 하단에서 별을 출력하는 조건문이 있는데, 이미 상단에서 공백을 출력했기 때문에 하단에 나오는 별은 빈자리를 채우는 형태로 나타난다.
```

**별 출력 예제(피라미드)**
```

코드 설명: 우측 정렬과 마찬가지로 공백을 출력하되, 층마다 i*2+1개의 별을 출력하는 형태이며, i*2+1개의 별을 출력하므로 중앙 대칭의 형태이다.

```

## 1. break문
반복문이나 switch문에서 현재 실행중인 코드를 즉시 종료하고 해당 코드 블록 다음의 코드로 제어를 전달한다.
->반복문에서의 break문을 통해서 해당 코드블록의 실행을 즉시 중단시킬 수 있다.
->switch문에서의 break문을 통해서 해당 case문을 즉시 중단 시킬 수 있다.

예시) 1.for문
for(;;){
...
    if(a)
        break; (바로 코드블록 탈출)
    }
    2. while문
while(){
...
 if(b)
     break; (else문을 안보고 바로 코드블록 탈출)
    else
    ...
}

    3. switch문
switch(){

    case value1:
        ...
    break; (바로 case1을 탈출 -> case2로 진행)

    case value2:
        ...
    break; (case2도 탈출)
}
    
```

### 2. continue문
반복문 내에서 사용되며, 현재의 반복 주기를 즉시 종료시키고 다음 반복 주기로 이동한다. 해당 반복주기에서 continue문 이후에 코드 실행없음

예시)
1. for 문
       for(초기화; 조건식; 증감식){
           ...
           if(a)
               continue;  (다시 증감식으로 올라가는 형태)
           printf("aa");
   }
2. while문
    while문(조건식){
        ...
        if(b)
        continue; //다시 조건식으로 가게해줌
    else
        ...
}
```

*** 확인문제 ***
1. **(break)**문은 특정 조건에서 반복은 완전히 중단할 때 사용하고, **(continue)**문은 특정 조건에서 해당 반복 주기를 건너뛸 때 사용한다.
